# Prompt: Auditoria Crítica (Framework Nova System - CAE)

Para finalizar nosso planejamento, ative o Critical Analysis Expert (CAE) do framework Nova System. 

Analise todo o plano financeiro e estratégico que construímos até agora. Quero que você procure agressivamente por falhas, riscos ocultos de estouro de orçamento ou gargalos operacionais que esquecemos. 

Seja um crítico implacável e forneça um plano de contingência para as duas maiores falhas que você encontrar no nosso projeto.
